JavaScript Elements

Click metrics

Pure JavaScript

document.querySelector('.click-image').onclick = function() {
  dataMetrics.sendToGA('Images', 'Click', 'Marlos Carmo');
}

jQuery

$('.click-image').click(function() {
  dataMetrics.sendToGA('Images', 'Click', 'Marlos Carmo');
});

Category: Images Action: Click Label: Marlos Carmo
